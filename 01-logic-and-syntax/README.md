# 01 - Logic and Syntax

Exercícios voltados aos fundamentos da linguagem, com foco em raciocínio algorítmico, estrutura básica de código e primeiros contatos com concorrência.

## Conteúdo

### `prime-generation/`
- `GeneratePrimes.java`: implementação clássica do crivo de Eratóstenes para gerar números primos até um limite informado.

### `concurrency/`
- `ex_Thread_e_Runnable.java`: exercício com `Runnable` e `AtomicInteger`, explorando execução concorrente e atualização compartilhada de estado.

### `reference/`
- `Lista_5.txt`: enunciados originais que deram origem a parte dos exercícios deste repositório.

## O que este bloco demonstra

- laços de repetição
- condicionais
- arrays
- funções auxiliares
- noções iniciais de concorrência
