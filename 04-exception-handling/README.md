# 04 - Exception Handling

Exercícios que enfatizam validação, tratamento de entradas inválidas e proteção contra cenários de erro previsíveis.

## Conteúdo

### `prime-validation/`
- `GeneratePrimesRefactored.java`: versão refatorada do gerador de primos, com validação do valor informado antes do processamento.

### `calculator-converter/`
- `Calculate.java`: enum de operações com divisão protegida contra divisão por zero.
- `Converter.java`: menu interativo com leitura de números e seleção de operação.
- `Main.java`: inicialização da aplicação.

### `phone-number-formatter/`
- `NumberFormatter.java`: leitura de entrada livre e tentativa de normalização de números de telefone.
- `Main.java`: execução contínua do formatador.

## O que este bloco demonstra

- validação de entrada
- tratamento de caso inválido
- mensagens de erro mais claras
- proteção contra falhas previsíveis
