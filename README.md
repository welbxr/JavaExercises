# Exercícios Java

Repositório organizado para estudo e revisão da linguagem Java, com exercícios separados por tema. A ideia é manter um histórico claro de prática, facilitar a navegação e deixar o material mais agradável para leitura por recrutadores e avaliadores técnicos.

## Estrutura

- `01-logic-and-syntax`: fundamentos da linguagem, algoritmos simples, laços, validações básicas e um exemplo com threads.
- `02-data-structures`: listas, igualdade entre objetos e um DAO genérico em memória.
- `03-oop-principles`: interfaces, enums, records e composição de comportamentos.
- `04-exception-handling`: tratamento de entradas inválidas, casos de borda e falhas previsíveis.
- `05-java-apis`: uso de APIs padrão como `String`, `Optional` e `Stream`.

## Como o material está organizado

Cada pasta principal contém um `README.md` explicando o foco dos exercícios daquela categoria e os arquivos agrupados por mini-projeto.

Os nomes originais dos exercícios foram preservados quando isso ajuda a manter o contexto do estudo.
