# 05 - Java APIs

Exercícios que exploram bibliotecas e recursos nativos do Java, como `String`, `Optional`, `Stream`, coleções e records.

## Conteúdo

### `string-extractor/`
- `AplicacaoStringExtrator.java`: exemplo de parsing de texto em formato JSON-like para estrutura de mapa.

### `optional-application/`
- `Main.java`: uso de `Optional` com transformação de objetos e valor padrão.
- `domain/User.java`, `domain/UserV2.java`, `domain/SexEnum.java`: modelos usados no exemplo.

### `stream-functions/`
- `domain/Main.java`: processamento de lista com `Stream`, `flatMap`, `filter`, `map` e `min`.
- `domain/User.java`, `domain/Contact.java`, `domain/ContactType.java`, `domain/Sex.java`: modelos do exercício.

## O que este bloco demonstra

- `String`
- `Map`
- `Optional`
- `Stream`
- `Comparator`
- `List`
